1.将看门狗2材质补充包.zip 和 XeKr红石显示1.19-V2.5材质补充包.zip 放入游戏目录下的材质包文件夹
2.将mod文件夹中所有文件放入游戏目录下的mod文件夹
其中【API】fabric-api-0.68.0+1.19.2.jar 和 【更多红石合成本体】extra-redstone-craft-fabric-1.19-17.0.5.jar 两个文件必须安装
其余文件可选装
3.credit.txt 包含了所有模组内使用的外部素材
4.util credit 包含了制作此模组使用的软件以及API

模组更新：你可以在此网站上查看最新的模组版本
https://www.curseforge.com/minecraft/mc-mods/extra-redstone-craft
